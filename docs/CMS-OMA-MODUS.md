# CMS Oma-Modus – Bedienung

## Login
1. `/admin/login` öffnen
2. E-Mail + Passwort eingeben
3. Dashboard öffnen

## Inhalte pflegen
- Im Dashboard die gewünschte Seite auswählen oder neu anlegen
- Bearbeiten:
  - Titel / Slug / Status
  - Hero Text + CTA
  - Farben
  - Panels (Kacheln)
  - Partnerblöcke

## Bilder hochladen
- Rechts im Medienbereich Datei auswählen
- Upload klicken
- Bild in Hero/Panel/Partner per Dropdown auswählen

## Live-Editor
- Im Dashboard auf **Live-Editor öffnen** klicken
- Auf der echten Startseite Inhalte direkt sehen und ändern
- Speichern klicken

## Veröffentlichen
- Status auf `published` setzen
- Speichern

## Sicherheit
- Admin-Zugangsdaten regelmäßig ändern
- Keine Passwörter in Doku oder Git speichern
